# 🏠 House Price Prediction using Machine Learning

## 📌 Overview

A Machine Learning project that predicts house prices based on property features using **Linear Regression**.

The model is trained on housing data, evaluated using regression metrics, and deployed using **Streamlit** to provide an interactive web application for real-time price prediction.

---

## 🚀 Features

* Predicts house prices based on user inputs
* Built using a supervised learning approach
* Interactive Streamlit web interface
* Trained ML model saved using Pickle

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Streamlit
* Jupyter Notebook

---

## 🤖 Machine Learning Workflow

```
Data Preprocessing
        ↓
Feature Selection
        ↓
Model Training
        ↓
Model Evaluation
        ↓
Streamlit Deployment
```

---

## 📊 Model Details

**Algorithm:** Linear Regression

**Dataset:** Housing dataset (5000 records)

**Evaluation Metrics:**

* Mean Absolute Error (MAE)
* R² Score
